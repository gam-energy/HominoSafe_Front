Make HTTPS trusted (required for Web Push)

TLS is already running. Chrome still shows Not Secure because the certificate is signed by SenioSentry’s private CA — not a public CA. Web Push and service workers need the browser to trust that CA.

Download SenioSentry CA

Desktop (Chrome / Edge)

  1. Open the downloaded seniosentry-ca.crt.
  2. Install it into Trusted Root Certification Authorities (Windows) or the system keychain and mark it trusted for SSL (macOS / Linux).
  3. Fully quit the browser, reopen it.
  4. Visit https://93.118.120.215:3000 — the padlock should no longer say Not Secure.
  5. Settings → enable Browser Web Push.

Android Chrome

  1. Download seniosentry-ca.crt on the phone.
  2. Settings → Security → Encryption & credentials → Install a certificate → CA certificate.
  3. Reopen Chrome and visit the HTTPS URL above, then enable Web Push.
